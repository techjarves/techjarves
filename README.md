# Tech Jarves

**Software Engineer & Tech Creator**  
*Building privacy-focused mobile, desktop, and system utilities.*

I develop open-source developer tools, custom Android ROM environments, and high-performance local AI clients. Through my YouTube channel, Tech Jarves, I share deep-dives into systems engineering, custom ROMs, and offline AI orchestration.

---

### Core Expertise

*   **Systems & Automation** — Custom Android ROM integration, Linux kernels, and shell automation.
*   **Offline Intelligence** — Porting C++ inference runtimes like llama.cpp and stable-diffusion.cpp to mobile platforms.
*   **App Architecture** — Cross-platform application engineering using Flutter, Dart FFI, and GetX.
